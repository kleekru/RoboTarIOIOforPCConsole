If marker (*) is displayed:

I/Space - insert selected chord
Top, Down - select other chord
T/click - edit current line text
Enter - add new line
V/Ctrl+Enter - add new verse

If chord is selected:

Left, Right - move through the chords
Top, Down - change current chord to another
D/Delete - remove the current chord
Space - move current chord 1 space to the right (if possible)
Backspace - move current chord 1 space to the left (if possible)
I - insert chord before current chord (then change it)
T/click - edit current line text
